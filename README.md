# Somatic Steve 2.0

## Overview
Somatic Steve 2.0 is a Discord bot designed to help users find information on the Somatic VR Tracker. It also serves as a test for a Google Sheets-based command generation system. **This project is not affiliated with SomaticVR as a company.**

## Features
- Guided somatic exercises
- Progress tracking
- Personalized recommendations
- User-friendly interface

## Installation
```bash
git clone https://github.com/yourusername/somatic-steve-2.0.git
cd somatic-steve-2.0
npm install
```

## Usage
```bash
npm start
```

## Requirements
- Node.js 14+
- npm or yarn
